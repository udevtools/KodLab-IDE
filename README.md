# 🎮 KodLab Console

> **Console de programmation virtuelle pour apprendre le développement de jeux**

KodLab est un environnement de développement éducatif qui permet d'apprendre la programmation en créant des jeux sur une console virtuelle de 400x300 pixels.

## ✨ Fonctionnalités

- **🎯 Langage .kod** - Syntaxe simple inspirée du JavaScript
- **🎮 Console intégrée** - Canvas 400x300px avec rendu temps réel
- **🎨 API graphique** - Rectangles, cercles, lignes, texte et couleurs
- **🎵 Audio** - Sons musicaux avec notes françaises (do, ré, mi...)
- **⌨️ Contrôles** - Gestion clavier (flèches, espace)
- **💾 Import/Export** - Sauvegarde et partage de projets .kod
- **🔧 Extension VSCode** - Coloration syntaxique pour l'IDE

## 🚀 Démarrage rapide

1. Ouvrez `index.html` dans votre navigateur
2. Écrivez votre code dans l'éditeur :
```kod
function start() {
    clearScreen("#001122")
}

function update() {
    drawText("Hello KodLab!", 150, 150, "#00d4aa")
}
```
3. Cliquez sur **🔨 Compiler** puis **▶️ Lancer**
